## 🎯 Objetivo do Projeto

Desenvolver uma aplicação web simples e intuitiva para cadastro e gerenciamento de lembretes do dia a dia, permitindo:

- Criar lembretes
- Definir data e hora
- Excluir tarefas
- Interface limpa e responsiva

---

## 🧩 Problema Resolvido

Muitas pessoas esquecem compromissos importantes como:

- Pagamento de contas  
- Tomar medicamentos  
- Reuniões  
- Entregas acadêmicas  

A aplicação LembreiFácil oferece uma solução simples e prática para organização pessoal.

---

## 🏗 Arquitetura da Solução

A aplicação segue uma arquitetura em três camadas:

- **Frontend:** HTML + CSS
- **Lógica:** JavaScript
- **Execução:** Navegador Web
